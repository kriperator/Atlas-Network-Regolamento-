# 🖥️ Protocollo Screenshare (SS)

### LINEE GUIDA PER IL CONTROLLO HACK

* **7.0.0 | FASE 1 - Il Freeze:** Quando si congela un utente sospetto in-game, il player ha esattamente **5 minuti di tempo** per accedere alla stanza vocale di controllo su Discord.
* **7.1.0 | FASE 2 - Ammissione Spontanea:** All'inizio del controllo, va chiesto al player se ha cheat da ammettere. In caso positivo, la sanzione viene dimezzata (7 giorni di ban invece di 14).
* **7.2.0 | FASE 3 - Ispezione:** Lo staffer può ispezionare solo ed esclusivamente i file inerenti a Minecraft (la cartella `.minecraft/mods`, i file `latest.log` e i software delle periferiche come Logitech o Razer). È vietato guardare file personali o social.
* **7.3.0 | FASE 4 - Rifiuto o Slog:** Se l'utente esce dal server durante il freeze o si rifiuta di collaborare, la sanzione è il `BAN` di 20 Giorni per controllo rifiutato.

{% hint style="success" %}
**7.4.0 | Esito Negativo:** Se l'utente risulta pulito, lo staffer deve sbannarlo/sbloccarlo immediatamente, scusarsi per il disturbo e augurargli buon proseguimento.
{% endhint %}
